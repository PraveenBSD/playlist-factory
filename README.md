# Playlist Factory

**Want to create a huge playlist?**
Sit back and watch **Playlist Factory** do the job.

### Configuration
- config.json
  ```
    {
        "playlistName": "Sample Playlist Name",
        "filePath": "/path/to/your/playlist/file",
        "chromeDriverRelativePath": "chromedriver",
        "userName": "user.name",
        "password": "user.password"
    }
  ```
